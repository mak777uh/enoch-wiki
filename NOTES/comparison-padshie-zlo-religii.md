# Сравнение подходов к падшим и злу в разных религиях

## Таблица сравнения

| Религия | Природа зла | Есть ли падшие? | Судьба зла | Выход/Спасение |
|---------|--------------|-----------------|------------|----------------|
| **Иудаизм** | Результат выбора ангелов | Да (Стражники) | Уничтожено на суде | Закон |
| **Христианство** | Активный противник Бога | Да (Сатана) | Вечный ад | Вера в Христа |
| **Шумер/Месопотамия** | Часть иерархии богов | Нет | Нет вечного суда | Нет концепции |
| **Индуизм** | Часть цикла кармы | Нет | Перерождение | Мокша |
| **Зороастризм** | Изначальный дух зла | Нет (изначально злой) | Будет побеждено | Следование Ахура-Мазде |
| **Гностицизм** | Онтологическая ошибка | Нет (демиург слеп) | Растворение в свете | Гнозис (знание) |
| **Буддизм** | Неведение и карма | Нет | Временное страдание | Нирвана |
| **Даосизм** | Дисбаланс | Нет | Растворится в Дао | Гармония с Дао |

## Типология моделей

### 1. Падение (иудаизм, христианство)

- Зло = результат выбора высших существ
- Есть «падшие» — ангелы, выбравшие зло
- Судьба: наказание или вечная борьба

### 2. Изначальное зло (зороастризм)

- Зло = один из двух изначальных принципов
- Нет «падших» — зло существует от начала
- Судьба: конечная победа добра

### 3. Циклическая модель (индуизм, буддизм)

- Зло = часть космического цикла
- Нет «падших» — есть состояния существования
- Судьба: выход через мокшу/нирвану

### 4. Онтологическая ошибка (гностицизм)

- Зло = невежество, а не моральный выбор
- Нет «падших» — есть слепой демиург
- Спасение через знание

### 5. Дисбаланс (даосизм)

- Зло = нарушение гармонии
- Нет «падших» — есть дисбаланс энергий
- Восстановление баланса

### 6. Иерархия богов (шумер)

- Зло = решение богов (потоп)
- Нет «падших» — нет морального падения
- Нет концепции спасения

## Ключевые выводы

1. **Модель зла определяется онтологией** — как религия видит устройство мира, так видит и зло

2. **Дуальность vs монизм** — зороастризм (дуальность) vs даосизм (монизм)

3. **Статика vs динамика** — зло либо изначально (зороастризм), либо результат процесса (индуизм)

4. **Падение — не универсально** — только в авраамических традициях есть концепция «падших»

5. **Карма меняет всё** — в буддизме и индуизме нет вечного зла — есть только цикл
