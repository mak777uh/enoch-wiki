# Автономный режим ведения вики

## Суть
При глубоком погружении пользователя в тему, агент обязан:
1. Самостоятельно создавать заметки (entity, analysis, comparison).
2. Обеспечивать связи (wikilinks + обратные ссылки).
3. Коммитить и пушить в GitHub.

## Алгоритм
- **Вход**: Пользователь высказывает идею / задает вопрос / углубляется.
- **Действие**: Агент создает узел в графе.
- **Выход**: Сообщение в чат "Created: [[note-name]]" + краткий вывод.

## Запрет
Не спрашивать "Создать заметку?". Действовать сразу.
