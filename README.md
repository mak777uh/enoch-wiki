# Enoch-Wiki

Личная база знаний в формате Obsidian-зеттелькастена.

## Что здесь главное

- `SOURCES/` - все исходные тексты в `.md`
- `NOTES/` - все заметки, summary, cards, entities, comparisons, analyses, guides и redirects
- `agent-basis-instructions.md` - главная инструкция для любых агентов

## Как устроена база

- `summary`-заметки - развёрнутые отчёты о прочтении целого источника
- обычные заметки - короткие атомарные узлы графа
- связи строятся через Obsidian wikilinks
- источники и заметки разделены, чтобы граф оставался чистым

## Что я исследую

- древние религии и мифологии
- падших ангелов, гигантов, потоп и демонологию
- эволюцию мотивов между традициями
- древнейшие письменные и реконструируемые устные слои
- храмы, пирамиды, мегалиты и большие сооружения
- сравнение культур по всему миру, без привязки к одному региону
